# CNN LSTM 
Implementation of CNN LSTM with Resnet backend for image sequence Classification


Put your image sequences dataset inside data/image_data
It should be in this form --
```
+ data 
    + image_data    
            - bowling
            - walking
            + running 
                    - running0
                    - running
                    - runnning1
                        - frame_0.jpg
                        - frame_1.jpg
                        - frame_2.jpg
                          ...
                        - frame_N.jpg
```
